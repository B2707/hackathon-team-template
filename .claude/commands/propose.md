---
description: File an idea or out-of-brief discovery (quarantined, FM triages)
allowed-tools: Bash
---
Run `./scripts/task propose $ARGUMENTS` (compose a crisp title if the
argument is rough).

Rules that make this safe:
- This is the ONLY sanctioned way to create an issue from a seat. Proposals
  land quarantined under the `proposed` label — invisible to the board until
  the First Mate promotes them.
- After proposing, RETURN TO YOUR BRIEF. Do not start the proposed work,
  do not expand scope, do not wait for a reply.
- If the proposal would change the plan itself (tool swap, feature add/cut,
  rescope), say so in the title with "PLAN:" — those route to Bader, always.
